Inserindo Videos com Hospedagem Local e Externa

Oque é Hospedagem local?

Hospedagem local HTML é o processo de hospedar um site ou aplicação web no computador pessoal, permitindo que seja criado e testado sem internet. Isso vale para arquivos do site como: imagens, video audios etc..

Oque é Hospedagem externa?

Hospedagem externa HTML é um serviço que armazena os arquivos de um site, incluindo os códigos HTML, em um servidor na nuvem. Isso permite que o site seja acessado na internet. Isso vale para arquivos do site como: imagens, videos, audios etc...
